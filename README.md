# Java Utilities
Some java utilities. <br>
This library requires Java 11 or newer.

[![Snapshot](https://repo.coding-stube.de/api/badge/latest/snapshots/dev/spoocy/utils/root?color=0F81C2&name=release&prefix=v)](https://repo.coding-stube.de/#/snapshots/dev/spoocy/utils/)

Be sure to replace the **VERSION** key below with the version shown above!

Common Utils:
```xml
<dependency>
  <groupId>dev.spoocy.utils</groupId>
  <artifactId>common-utils</artifactId>
  <version>VERSION</version>
</dependency>
```

```xml
<dependency>
  <groupId>dev.spoocy.utils</groupId>

    <version>VERSION</version>
</dependency>
```
